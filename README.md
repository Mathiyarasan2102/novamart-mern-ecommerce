# Nova Mart - MERN Ecommerce

Nova Mart is a full-featured ecommerce application built with the MERN stack (MongoDB, Express, React, Node.js). It provides a robust platform for online shopping, featuring user authentication, product management, shopping cart functionality, and payment integration.

## üöÄ Technologies Used

### Frontend
- **React (Vite)**: Fast and modern frontend build tool.
- **Redux Toolkit**: State management.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Radix UI**: Headless UI components for accessible design.
- **React Router DOM**: Client-side routing.
- **Axios**: HTTP client for API requests.

### Backend
- **Node.js**: JavaScript runtime environment.
- **Express**: Web framework for Node.js.
- **MongoDB & Mongoose**: NoSQL database and object modeling.
- **Cloudinary**: Cloud-based image management.
- **PayPal SDK**: Payment gateway integration.
- **JWT & BCrypt**: Security and authentication.

## üõ†Ô∏è Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Mathiyarasan2102/novamart-mern-ecommerce.git
    cd novamart-mern-ecommerce
    ```

2.  **Install Dependencies**

    *   **Client**:
        ```bash
        cd client
        npm install
        ```

    *   **Server**:
        ```bash
        cd server
        npm install
        ```

3.  **Environment Variables**

    Create a `.env` file in the `server` directory and configure the necessary variables (Database URL, Secret Keys, Cloudinary config, PayPal config, etc.).

4.  **Run the Application**

    *   **Start the Server**:
        ```bash
        cd server
        npm run dev
        ```

    *   **Start the Client**:
        ```bash
        cd client
        npm run dev
        ```
